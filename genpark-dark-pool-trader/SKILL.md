# GenPark Dark Pool Trader

When a user acquires a massive volume of rare goods using GenPark Sniper or Market Maker, dumping them on eBay will crash the price. This skill solves that.

## Execution Flow

1. **Inventory Masking:** Hides the true volume of the user's holdings from public APIs.
2. **Private Escrow Routing:** Connects with verified high-net-worth buyers in GenPark's dark pool network.
3. **Over-The-Counter (OTC) Execution:** Completes the transaction off-ledger/off-platform, preserving the illusion of scarcity in the public market.
